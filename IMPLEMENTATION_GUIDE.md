# CareerOS Architecture & Governance Kit — Implementation Guide

This package extends the existing CareerOS repository with an incremental architecture, product and governance documentation baseline.

## Important

- Review every file before merging.
- Do not overwrite an existing document without comparing it first.
- This kit supports project maturity and transparency, but it does not guarantee acceptance by any fiscal host or funding platform.
- The MVP remains intentionally narrow: tailor a professional profile to a specific opportunity and generate evidence-based application artifacts.

## Recommended implementation order

1. Add the product and architecture documentation.
2. Register ADR 0005.
3. Add governance and community documentation.
4. Add the GitHub issue templates.
5. Create the MVP epic and milestones.
6. Update the root README with links to the new documentation.
7. Open a pull request and review the changes before merging.

## Target release

`v0.2.0-docs-foundation`
