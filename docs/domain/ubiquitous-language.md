# Ubiquitous Language

| Term | Meaning |
|---|---|
| Professional Profile | Structured representation of a person's professional identity |
| Evidence | Source that supports a professional claim |
| Claim | Statement about experience, skill, result or qualification |
| Opportunity | Job, contract, program, grant or professional objective |
| Requirement | Expected qualification, responsibility or constraint |
| Tailoring | Selecting and adapting approved evidence for an opportunity |
| Application | Context connecting a profile and an opportunity |
| Artifact | Generated résumé, cover letter, report or response |
| Provenance | Trace of where a claim originated and how it was transformed |
| Match | Degree of evidence-backed alignment with an opportunity |
| Gap | Opportunity requirement without adequate supporting evidence |
| Verification | State indicating how strongly evidence supports a claim |
| Approval | Explicit user acceptance of a generated artifact |
