# Governance

CareerOS should evolve as an open source project connected to SaaSCoop and FreireAI.

## Roles

- Maintainer: responsible for releases and project direction
- Contributor: contributes code, documentation, tests or examples
- Reviewer: validates quality and architecture alignment
- User: uses CareerOS to manage career identity
- Sponsor: financially supports the project without controlling it

## Decision making

Important architectural decisions must be recorded as ADRs.
