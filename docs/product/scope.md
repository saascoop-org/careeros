# Product Scope

## In scope for the initial product

- structured professional profile;
- structured opportunity profile;
- evidence and provenance;
- opportunity analysis;
- experience and skill ranking;
- résumé composition;
- ATS-oriented keyword coverage;
- output validation;
- Markdown and PDF export;
- audit trail.

## Future scope

- external profile connectors;
- document ingestion;
- knowledge graph;
- semantic search;
- portfolio generation;
- LinkedIn and GitHub optimization;
- interview preparation;
- certification and learning recommendations;
- optional ecosystem integrations;
- MCP server and public SDK.

## Non-goals

CareerOS is not:

- an applicant tracking system;
- a recruitment marketplace;
- an automatic job application bot;
- a source of fabricated credentials;
- a replacement for user judgment;
- an educational platform;
- a mandatory component of FreireAI.
