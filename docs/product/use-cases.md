# Core Use Cases

## UC-001 Create or update a professional profile

The user records structured experiences, skills, achievements and evidence.

## UC-002 Register an opportunity

The user provides a job description, client request, program call or professional objective.

## UC-003 Analyze opportunity relevance

CareerOS extracts responsibilities, technologies, seniority, language expectations and important keywords.

## UC-004 Rank professional evidence

CareerOS ranks experiences, projects and achievements according to the opportunity while preserving source traceability.

## UC-005 Generate a tailored résumé

CareerOS composes an editable résumé from ranked, approved evidence.

## UC-006 Validate output

CareerOS checks unsupported claims, duplicated content, missing evidence and keyword coverage.

## UC-007 Export the application package

The user exports Markdown, PDF and audit metadata.

## UC-008 Review and approve

The user edits and explicitly approves the final content before external use.
