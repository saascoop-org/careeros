# Case Study — International SME Application

## Purpose

Validate the CareerOS Opportunity Tailoring Engine using a real international application.

## Opportunity profile

The opportunity involves technical review, course development support and content validation across Microsoft and cloud technologies.

Priority areas include:

- Microsoft Fabric;
- Power BI;
- SQL Server;
- Azure Data Analytics;
- data engineering;
- technical documentation;
- process and workflow validation;
- clear technical communication in English.

## Candidate evidence categories

The profile should prioritize verifiable evidence related to:

- enterprise business intelligence;
- SQL Server and SSIS;
- Power BI;
- Microsoft Fabric;
- ETL and data platform modernization;
- technical documentation;
- validation of workflows and requirements;
- public technical content;
- certifications and learning paths;
- experience communicating with technical and business stakeholders.

## Expected outputs

- one-page or two-page international résumé;
- match analysis;
- supported keyword list;
- evidence map;
- concise professional summary;
- optional Upwork proposal and interview notes.

## Privacy

The public repository should use anonymized or synthetic sample data unless the candidate explicitly approves publication of personal information.
