# ADR 0001: Record architecture decisions

## Status

Accepted

## Context

CareerOS is expected to evolve into a modular open source platform with many contributors, integrations and agents.

## Decision

We will use Architecture Decision Records under `docs/adr` for significant technical and governance decisions.

## Consequences

- Decisions become auditable.
- Contributors understand why choices were made.
- The project avoids architecture drift.
