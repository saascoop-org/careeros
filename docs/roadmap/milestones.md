# Proposed Milestones

## M0 — Documentation Foundation

Exit criteria:

- vision, mission and principles published;
- MVP scope accepted;
- architecture documented;
- ADR 0005 accepted;
- roadmap published;
- governance process documented.

## M1 — First Tailored Résumé

Exit criteria:

- profile schema;
- opportunity schema;
- evidence schema;
- CLI generation command;
- Markdown résumé;
- automated tests;
- real Upwork application generated.

## M2 — Validated Application Package

Exit criteria:

- PDF export;
- keyword analysis;
- gap analysis;
- evidence map;
- claim validator;
- user review checklist.

## M3 — Reusable MVP

Exit criteria:

- second opportunity processed without code changes;
- public example with anonymized data;
- contributor setup under 15 minutes;
- release `v0.1.0`.
